# Desafio Supremo

![desafio-supremo.gif](images_readme/desafio-supremo.gif)

App criado para mostrar uma lista de extrato bancário consumida de uma API externa. 
Com a ideia de praticar os conhecimentos obtidos até o momento.

## Recursos utilizados

- MVVM
- Retrofit
- Hilt
- Corroutines
- RecyclerView
- ViewBinding
